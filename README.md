## introduce
국민대학교 논리회로설계 강의 시간 중, tabular method를 배우게 되었습니다.
이때, 강의 후 과제로 만들게 된 tabular method cli 소스입니다.

## usage
main.cpp를 g++ 컴파일러로 컴파일/실행해주세요.
